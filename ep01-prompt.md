# EP01 Prompt — 旋转六边形弹球

> **系列**：model-battle 01
> **性质**：网红测试题，2025 年初 LLM 圈爆火
> **特点**：碰撞检测 + 旋转坐标系下的反射向量 + 重力+摩擦真实感

---

## 题目

**旋转的正六边形里关着一个小球，受重力和摩擦力影响，必须能真实地从正在旋转的边上反弹。**

## 为什么选这道题

- 网红测试题，自带认知落差（一听就有共鸣）
- 区分度极大：碰撞检测、旋转坐标系下的反射向量、重力+摩擦的真实感
- 单 HTML 双击可跑，不需要环境
- 视觉冲击最强：球弹来弹去本身就是看点

## 统一 prompt

```
用 HTML + Canvas + JavaScript 写一个程序：

一个旋转的正六边形里关着一个小球。
小球受重力和摩擦力影响，必须能真实地从正在旋转的边上反弹。

要求：
1. 只输出一个完整的 HTML 文件
2. 不需要任何外部依赖（不要 CDN、不要装库）
3. 保存为 .html 后直接双击就能在浏览器里运行
4. 物理效果要真实
5. 视觉要好看
6. 把 <title> 标签设置为「你的模型名和版本」，方便我同屏对比时区分（例如「DeepSeek V4 Pro」「Kimi K2.6」）
7. 在 HTML 第一行用注释也标一下模型名（例如 <!-- Model: DeepSeek V4 Pro -->），方便我保存文件时命名

文件命名为 buchong
```

## 命名约定（保存 HTML 时）

```
deepseek-v4.html
kimi-k26.html
mimo-v25.html
glm-51.html
qwen-36.html
minimax-m27.html
```

## 评分 5 维度

- 能直接运行（保存 HTML 双击打开不报错）
- 物理真实（重力下落不是匀速，摩擦让球逐渐减速）
- 反弹方向（从旋转的边反弹后角度对）
- 视觉好看
- 模型名标注（title + 第一行注释）
