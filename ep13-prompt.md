# 任务

生成一个**单文件 HTML**，实现一个完整的拖拽看板（Kanban Board）。

## 硬指标

### 1. 列（Columns）

- 3 个默认列：待办（To Do）/ 进行中（In Progress）/ 已完成（Done）
- 支持新增/删除列
- 列标题可编辑
- 每列显示卡片数量

### 2. 卡片（Cards）

- 每个列内可新增卡片
- 卡片内容：标题（必填）+ 描述（可选）+ 标签（可选）
- 卡片可编辑（点开弹窗修改）
- 卡片可删除（确认弹窗）

### 3. 拖拽（Drag & Drop）

- 卡片可在同列内拖拽排序
- 卡片可拖拽到其他列
- 拖拽时有视觉反馈（卡片跟随 / 占位提示）
- 支持触摸设备（touch events）

### 4. 数据持久化

- 所有数据保存到 localStorage
- 刷新页面后恢复全部状态

### 5. 视觉

- 现代简洁风格（参考 Notion / Trello）
- 卡片标签用不同颜色区分
- 响应式：桌面 3 列并排，移动端单列堆叠
- 空状态提示（当列中没有卡片时）

## 约束

- **单文件 HTML**，所有 CSS 和 JS 内嵌
- **无外部依赖**（不引用 CDN、不引用外部库）
- 输出文件保存到工作区目录，文件名 `index.html`
