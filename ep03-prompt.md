```
用 HTML + JavaScript + p5.js（WEBGL 模式）写一个程序：

一个 3D 中国九连环（Chinese Linked Rings / 九连环），自动播放完整解开过程。

【几何结构】
- 一根水平矩形柄板（约 40cm × 5cm × 1cm 比例），板的中线上等距开 9 个长方形通孔
- 9 个圆形金属环（编号 1~9，1 在最右/最外，9 在最左/最里）
- 相邻环互相穿过对方：环 i 穿过环 i+1 的圆形开口（这是九连环的核心拓扑）
- 每个环有两种状态：
  - "上"：环穿过柄板上对应通孔，挂在板上
  - "下"：环从板上脱出，挂在前一个环（i-1）上

【解开规则 — 必须严格遵守】
- 第 1 环（最外）可独立取下或装上
- 第 i 环（i ≥ 2）：当且仅当 第 i-1 环为"上"，且 第 1 ~ i-2 环全部为"下" 时，才能取下或装上
- 9 环全部解开需要恰好 341 步（这是数学定理，可验证）

【动画与界面】
- 程序加载后自动播放完整 341 步动画
- 每步动画 0.4-0.8 秒，操作的环用高亮色标出
- 屏幕固定显示：「当前步数 / 总步数」(如 "127 / 341") + 「已解开环数」(如 "已解 4 / 9")
- 鼠标拖拽可旋转 3D 视角

【硬性要求】
1. 只输出一个完整的 HTML 文件
2. 唯一允许的外部依赖：p5.js（CDN 引入）
3. 保存 .html 后双击浏览器就能跑，不报错、不白屏
4. 几何拓扑必须正确：环和环、环和板的穿过关系不能错
5. 算法必须正确：每步都合规，总步数 = 341
6. 3D 透视渲染，能看出立体感
7. 视觉要好看（颜色、光影、阴影任选）
8. <title> 设为你的模型名和版本（如「DeepSeek V4 Pro」）
9. HTML 第一行用注释标模型名（如 <!-- Model: DeepSeek V4 Pro -->）

文件命名为 (你的序号)-jiulianhuan

```

## 数学验证（备查）

九连环解开步数公式：

```
a(n) = (2^(n+1) - 2) / 3   （n 为偶数）
a(n) = (2^(n+1) - 1) / 3   （n 为奇数）
```

或递推：`a(n) = 2·a(n-2) + a(n-1) + 1`

| n | 步数 |
|---|-----:|
| 1 | 1 |
| 2 | 2 |
| 3 | 5 |
| 4 | 10 |
| 5 | 21 |
| 6 | 42 |
| 7 | 85 |
| 8 | 170 |
| **9** | **341** |
