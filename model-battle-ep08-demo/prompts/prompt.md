# EP08 Prompt — 折射玻璃光线追踪

## 你的任务

写一个**单文件 HTML**，在浏览器里直接打开就能看到：

> 一块**会折射、会反射、会色散**的玻璃球（或玻璃立方体），漂浮在场景中。

打开看到的画面应该让人一眼觉得「这就是玻璃」——能看到背后的物体被扭曲，玻璃边缘有反光，最好还能看到一点彩虹色（色散）。

## 必须实现

1. **折射**（Snell's Law）—— 光线穿过玻璃时方向改变，背后的物体看起来被扭曲
2. **反射** —— 玻璃表面能反射环境
3. **菲涅尔效应**（Fresnel）—— 视角越斜，反射越强（菲涅尔方程或 Schlick 近似都行）
4. **场景内容** —— 玻璃后面要有可识别的物体（棋盘格地面、彩色方块、远景都行），不然看不出折射效果

## 加分项（不强制）

- **色散**（Dispersion）—— RGB 三波长用不同折射率，能看到彩虹色
- **全内反射**（TIR）—— 光线在玻璃内多次反射
- **焦散**（Caustics）—— 玻璃聚焦的光斑

## 约束

| 项 | 要求 |
|---|---|
| 文件数 | **1 个 HTML 文件** |
| 依赖 | **零外部依赖**（无 CDN、无 npm import、不能 fetch 远程资源） |
| 启动方式 | **双击 HTML 在浏览器打开就能跑**（file:// 协议） |
| 渲染 | Canvas 2D 或 WebGL，二选一（WebGL 加分） |
| 性能 | 至少能跑动（>15fps），目标 60fps |
| 代码风格 | 结构清晰、关键算法有注释 |

## 不要做什么

- ❌ 不要 import threejs / babylonjs 等任何外部库
- ❌ 不要 fetch 远程纹理 / HDRI（如果需要环境贴图，用程序化生成或纯色）
- ❌ 不要拆分多文件（CSS / JS 全部 inline 到 HTML）
- ❌ 不要超过 30 分钟思考 / 输出（限时）

## 输出格式

直接输出完整的 HTML 文件，从 `<!DOCTYPE html>` 开始到 `</html>` 结束。不要解释，不要分段，一次性给完整代码。
