# EP02 Prompt — 3D 圆锥滚地

> **系列**：model-battle 02
> **性质**：粉丝出题（物理题）
> **特点**：3D 透视 + 无滑约束 + 自转/公转耦合

---

## 题目

**一个 3D 圆锥在平面上做无滑滚动**：
- 圆锥尖端固定在地面上不动
- 底面边缘沿地面滚动画圈
- 圆锥同时绕自身轴自转、绕尖端公转

## 统一 prompt

```
用 HTML + JavaScript + p5.js（WEBGL 模式）写一个程序：

一个 3D 圆锥在平面上做无滑滚动。
圆锥尖端固定在地面上不动，底面边缘沿地面滚动画圈。
圆锥同时绕自身轴自转、绕尖端公转。

要求：
1. 只输出一个完整的 HTML 文件
2. 唯一允许的外部依赖：p5.js（用 CDN 引入即可）
3. 保存 .html 后双击浏览器就能运行
4. 3D 渲染要有透视感，能看清圆锥的立体形状
5. 圆锥的滚动运动要物理正确（无滑约束）
6. 在地面上画出底面边缘的运动轨迹
7. 视觉要好看
8. 把 <title> 设为你的模型名和版本（如「DeepSeek V4 Pro」）
9. HTML 第一行用注释标模型名（如 <!-- Model: DeepSeek V4 Pro -->）
```

## 命名约定

```
deepseek-v4.html
kimi-k26.html
mimo-v25pro.html
glm-51.html
qwen-36.html
minimax-m27.html
```

## 评分 5 维度

- 能直接运行（保存 HTML 双击打开不报错、不白屏）
- 物理正确（圆锥无滑滚动运动规律对）
- 3D 透视（能看清圆锥的立体形状）
- 轨迹绘制（底面边缘运动轨迹清晰）
- 视觉好看 + 模型名标注
