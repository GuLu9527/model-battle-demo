# 任务：粒子流体仿真（双液体 · 单文件 HTML）

写一个粒子流体仿真，两种液体在一个**半圆碗**里流动混合。

## 物理要求

- 用 **SPH（Smoothed-Particle Hydrodynamics）** 算法
- **两种液体**同屏：水（蓝色）+ 蜡（橙色）
  - 蜡比水**轻**且**粘度更高**
  - 搅完之后蜡应该浮上来，水沉下去
- **重力** + **鼠标搅拌**（鼠标移动时对附近粒子施加力）
- 至少 **500 个粒子** 流畅运行（≥ 30 FPS）

## 容器

- **半圆碗**（不是矩形盒）
- 粒子触碰碗壁反弹，不能穿透

## 硬性要求

1. **单文件 HTML**，零外部依赖（不准引 CDN / Three.js / D3 等）
2. 仅允许浏览器内置 API（Canvas2D 或 WebGL）
3. **双击 .html 文件**就要能跑（`file://` 协议）
4. 左上角显示 **FPS** 和 **粒子数**

## 输出格式

只输出一个完整的 `index.html` 文件，开头 `<!DOCTYPE html>`，结尾 `</html>`。
不要任何额外说明文字、不要 markdown 代码块包裹。
