# EP07 Prompt

你打开的文件 `boids-buggy.html` 是一个 **3D 鸟群模拟程序**。

用浏览器打开它，你会发现行为明显不对。

## 这个程序应该做什么

**Boids**（Craig Reynolds, 1987）用三条规则模拟鸟群自发形成群落：

- 靠太近时互相排斥，保持间距
- 逐渐与周围邻居对齐飞行方向
- 向邻居中心靠拢，形成群落

正常情况下，你会看到：鸟不相互拥挤，会自然聚集成群，朝同方向飞，飞出边界从对面出现，速度稳定。

## 你的任务

找出代码里所有的 bug，修复到行为正确为止。

## 约束

- `project()` 和 `draw()` 函数上方有注释 `do not modify`，不要动它们
- 不要引入任何外部库或 CDN
- 修完之后必须能在浏览器里直接打开运行
